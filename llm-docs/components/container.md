# CDSContainer

Import from `@ciscodesignsystems/cds-react-container`.

See [README](../../docs/components/container-README.md) for full props and usage.

Simple container with padding and optional border/shadow for grouping content sections.

```tsx
<CDSContainer>
  <CDSHeading size="sub-section">Section Title</CDSHeading>
  <CDSText>Content here</CDSText>
</CDSContainer>
```

- For elevated, interactive content blocks, use `CDSCard` instead.
